# AI Powered Cricket Highlight Generator
### Intelligent Multimodal Cricket Highlight Generation System
**Submitted to:** HCL Technologies  
**Role:** Network Engineering Trainee  
**Date:** March 2026

---

## What This Project Does

This system automatically generates a 10-minute cricket highlight reel 
from a full match using artificial intelligence. Instead of a human editor 
watching hours of footage, the AI analyses every ball bowled using 8 
independent signal detectors and selects only the most exciting moments.

---

## The Problem It Solves

Manual cricket highlight generation takes 3-5 hours per match and suffers from:
- False positives — crowd reactions and fireworks misidentified as game events
- Missing temporally important moments — e.g. a dropped catch that gains 
  importance only after the batsman scores a century
- Redundant clips — same type of event repeated multiple times
- Poor context interpretation — a boundary in a last-ball finish is far more 
  exciting than the same boundary in a comfortable win
